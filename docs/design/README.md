# Diseño del Módulo

Esta sección contiene todos los entregables relacionados con el diseño técnico del módulo asignado.

## Estructura

```
design/
├── containers/              # D1.1 Diagrama de Contenedores
│   └── template-containers.md
├── components/              # D1.2 Diagrama de Componentes
│   └── template-components.md
├── code/                    # D1.3 Diagrama de Código
│   └── template-code.md
└── database/                # D1.4 Diseño de Base de Datos
    └── template-database.md
```

## Entregables

### D1. Modelo de Diseño
- **D1.1 Diagrama de Contenedores:** Arquitectura de contenedores (C4 nivel 2)
- **D1.2 Diagrama de Componentes:** Componentes internos (C4 nivel 3)
- **D1.3 Diagrama de Código:** Estructura de código (C4 nivel 4, si es necesario)
- **D1.4 Diseño de Base de Datos:** Modelo entidad-relación

## Cómo usar esta estructura

1. **Copiar templates:** Usar los archivos template-* como base
2. **Desarrollar diagramas:** Usar Mermaid para diagramas C4 y entidad-relación
3. **Mantener actualizado:** Revisar y actualizar según evolucione el diseño
4. **Versionar cambios:** Usar commits descriptivos para cada entregable

## Herramientas Recomendadas

- **Mermaid:** Para diagramas C4 y entidad-relación
- **GitHub:** Renderizado automático de diagramas Mermaid
- **VS Code:** Extensión Mermaid para edición local
